# Moments, Retold (https://moments-retold.vercel.app)

A small experiment in turning images into narrated stories.

Upload a photo, generate a short story, and hear it spoken aloud.

## What it does

- Upload an image  
- Choose a storytelling tone (nostalgic, cinematic, funny)  
- Generate a short story inspired by the image  
- Listen to the narration  
- Download text, audio, or a simple video version  

## Why this exists

Photos are everywhere—but they’re silent.

This project explores how voice can turn static images into something more emotional, more interpretable, and more shareable.

## Stack

- Next.js (App Router)
- OpenAI (image → story)
- ElevenLabs (text → voice)

## Notes

- Stories are intentionally framed as *one possible interpretation*  
- The focus is on experience, tone, and pacing—not just functionality
